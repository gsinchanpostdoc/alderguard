AlderIPM-Sim — Alnus Ecoepidemic Decision-Support

An in-browser decision-support tool for the black alder (Alnus glutinosa) – alder leaf beetle (Agelastica alni) – tachinid parasitoid (Meigenia mutabilis) – insectivorous-bird ecoepidemic system. Pick any of 41 Natura 2000 alluvial-forest sites across Europe on the interactive map, calibrate 23 model parameters from a continuously-updated literature metadatabase, run multi-year simulations, detect early-warning signals of regime shifts, fit to your own field observations, and compare integrated pest-management strategies. Every parameter slider carries a literature-pooled 95 % confidence interval and citation trail. Companion Python package and R package share the same model and outputs.

Open Application Source code & packages
Cite: Ghosh, S. et al. (2026). AlderIPM-Sim: Hybrid seasonal-annual modelling and early warning signals for the Alnus-beetle-parasitoid-bird ecoepidemic system. Frontiers in Ecology and Evolution (manuscript submitted).
